# DevOps Utility Hub

A collection of 12+ essential tools for DevOps engineers, built with pure HTML, CSS, and JavaScript. Zero backend, 100% client-side.

## 🚀 Available Tools

- **YAML:** Validator, Formatter, and YAML ↔ JSON Converter.
- **Docker:** Dockerfile and Docker Compose generators.
- **Security:** Base64 Encoder/Decoder, Password Generator, and JWT Decoder.
- **Network:** Subnet Calculator and CIDR range finder.
- **Utility:** Cron Expression Builder, Regex Tester, and .gitignore Generator.

## 🛠️ How to Run

1. Open `index.html` in any browser.
2. OR run a local server:
   ```bash
   python -m http.server 8080
   ```
